# ChatGPTクローン — 全体像と完成イメージ（共有会用1枚）

## 一言でいうと
「ChatGPTがなぜああ動くのか」を、ミニChatGPTを自作して体で理解するプロジェクト。

## 最後まで進むと出来上がるもの
- Googleログイン付きの**自分専用チャットアプリ**
- 返事が**1文字ずつ流れてくる**（ストリーミング、第11章）
- **会話が複数持てて、過去の会話を再開できる**（第5章）
- APIキー漏洩対策と使いすぎ防止つき（第2章・第6章）
- 長い会話でも壊れない（要約＋ログ、第8-9章）

## 仕組みの全体像（登場人物は3人＋倉庫）

```
ブラウザ ──質問──▶ 自分のサーバー ──鍵＋履歴ごと──▶ LLM(OpenAI)
（画面）◀─返事表示── （鍵と履歴の番人）◀────返事──────
                        │
                        ▼
                     Supabase（会話ログ・ログイン。後日）
```

## 背骨2本（これが学びの芯）
1. **LLMは毎回忘れる** — API呼び出しはステートレス。「会話が続いてる感」は、こちら側が毎回全履歴を送り直して作っている。
2. **鍵はサーバーの外に出さない** — APIキー＝財布。ブラウザに置いたら盗まれる。自分のサーバーだけが鍵を持ち、使用量も管理する。
